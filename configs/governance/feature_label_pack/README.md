# Feature/Label Study Input Pack Templates

This directory contains synthetic templates for `StudyInputPack` records.

The template shows only the shape of the governed input bundle consumed by
future study tooling: `freq_` FeatureRequest handles, `lspec_` LabelSpec
handles, one `aspec_` AlphaSpec handle, and a `dataset_scope` mapping aligned
with `StudySpec.dataset_scope`.

The handles are placeholders over synthetic metadata. They are not real
research ids, not registry evidence, and not proof that any study was run or
accepted.
