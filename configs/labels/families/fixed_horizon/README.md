# Fixed-Horizon Label Family Config

This directory is reserved for declarative selections of the FLF-P17
fixed-horizon label family.

The family covers trade-close forward returns and midprice forward returns over
1m, 3m, 5m, 10m, and 30m horizons. Config files here must not contain
materialized label values, raw provider references, registry state, broker/live
behavior, or research claims.
