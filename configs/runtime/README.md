# Runtime Config Placeholder

`configs/runtime/` is the committed home for future Research Runtime
configuration templates. RT-P02 establishes the directory only.

Future files may describe local, deterministic runtime plans, cache policy, or
report formatting defaults after their owning phases define the contracts.
Do not place secrets, credentials, provider responses, local data paths,
runtime values, heavy artifacts, database files, logs, caches, or machine-local
state in this directory.
