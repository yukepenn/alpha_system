# Cross-Market Diagnostics Runtime Config

This directory contains RT-P10 synthetic-safe configuration scaffolding for
`alpha_system.runtime.diagnostics.cross_market`.

The defaults are descriptive thresholds for ES/NQ/RTY alignment, coverage,
timestamp synchronization, lead/lag, and regime-conditioned correlation
summaries. They do not authorize raw provider access, external calls, broker
activity, paper or live trading, deployment, strategy construction, or promotion
decisions.
