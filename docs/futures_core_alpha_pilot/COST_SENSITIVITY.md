# Cost Sensitivity

`FUTCORE-P21` consolidates the cost-stress and thin-session-stress diagnostics
from `FUTCORE-P16` through `FUTCORE-P20`.

Primary report:

- `research/futures_core_alpha_pilot_v1/cost/cost_sensitivity_report.md`

The consolidation is value-free research evidence. It records profile coverage,
runtime diagnostic statuses, cost-stress metadata, thin-session penalty metadata,
and gaps only. `zero_cost` is diagnostic-only policy context and is never a
survival, promotion, watch, candidate, paper/live, broker, deployment, or
capital-allocation basis.

Key carried limitations:

- Four cross-market ideas have zero-fill cost-stress reports because P16
  rejected the required cross-market legs before populated cost stress.
- BBO-unavailable fallback and slippage/capacity proxy language remain visible
  across completed cost reports; no capacity or tradability proof is claimed.
- Thin-session-only or thin-session/RTH-comparator gaps remain diagnostic
  limitations for later matrix and evidence-ledger phases.
