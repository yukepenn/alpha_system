# SHIP_REFIT_V1 Evidence Scaffold

This directory is a value-free placeholder for commit-eligible evidence notes
and indexes created by later SHIP_REFIT_V1 phases.

Allowed content is limited to reviewable summaries, check summaries, and
value-free provenance indexes. Do not commit raw data, canonical data, feature
or label values, provider responses, local-only run artifacts, databases,
Parquet/Arrow/Feather files, logs, caches, secrets, or market-performance
claims.
