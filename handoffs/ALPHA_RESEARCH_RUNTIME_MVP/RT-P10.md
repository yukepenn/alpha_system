# RT-P10 Handoff - Cross-Market Diagnostics Runtime

## Scope Completed

Implemented `alpha_system.runtime.diagnostics.cross_market` as a descriptive
Cross-Market ES/NQ/RTY diagnostics runtime. The runtime consumes resolved
`RuntimeInputPack` handles plus caller-supplied in-memory diagnostics
observations, enforces exact-timestamp / `available_ts` alignment, and emits a
scalar-only `CrossMarketDiagnosticsReport` plus visible `DiagnosticsRunRecord`.

The runtime delegates pair, lead/lag, and regime-conditioned correlation
summaries to `alpha_system.research.correlation.correlations_to_existing_factors`.
It adds no Pearson/rank/cross-market correlation implementation. Spread/residual
sections are scalar arithmetic descriptors only and make no strategy or
portfolio object.
